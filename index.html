  For this challenge we were tasked with researching, defining, and creating a solution to an issue related to one of the Global Grande Challenges.   In this project we individually designed a solution for a problem using biomimicry. My project looked at similarities between bee hive traffic control and ours, and how some of their designs could be implemented in our urban systems. The full process can be found on my other Spark webpage.
